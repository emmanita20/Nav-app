## 🚀 Getting Started

### 1. Clone & Install
```bash
cd navi-app
cd backend && npm install
cd ../frontend && npm install
```

### 2. Configure
```bash
cd backend
cp .env.example .env
# Edit .env with MongoDB URI, API keys
```

### 3. Run
```bash
# Terminal 1: Backend
cd backend && npm run dev

# Terminal 2: Frontend
cd frontend && npm start
```

### 4. Test
- Visit http://localhost:3000
- Sign up or login with demo credentials
- Enable location permission
- Test navigation & emergency features
